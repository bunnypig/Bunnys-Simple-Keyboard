# Bunny's Simple Keyboard
*A virtual keyboard for Linux with Wayland support and extensive customization options.*


This is a fork of https://github.com/mdev588/vboard with Swedish SE layout and a couple of other changes like key modifiers. I've also added some kWin rules to make the keyboard always be above other apps and not accept focus.
This is for personal use and all credits goes to @mdev588 . I will not include other languages and only keep it for Swedish SE and only update it if anything breaks.

How to install:

1. Add the COPR repository
   
   dnf copr enable bunnypig/bunnys-simple-keyboard

3. Run install command
   
   dnf install bunnyskeyboard
